Let's Nolja

A game site about increasing knowledge and having fun

 Hangman A game where you have to guess one word, by choosing letters. If these letters are in the word, you get another chances until you find the word. If you fail, you lose one life. If you lose all your lifes, you lose (obvious right ?). Quiz A quiz where you get asked all types of questions. From cultural ones to funny ones. Find the differences A game where you have to find the differences between pictures. Sudoku Work in progress...
